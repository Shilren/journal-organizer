# 看板使用说明

三个看板（[[本周看板]]、[[本月看板]]、[[总览看板]]）会**实时**从你各分类文件夹里的当日笔记中读取 `## 时间 · 标题` 条目并聚合——不用给每条记录加任何标注，也不用手动更新，打开就是最新的。**分类自动发现**：凡是「文件夹里有 `YYYY-MM-DD.md` 文件」的文件夹都会被当作一个分类，颜色按字母序自动分配。

## 安装到你的仓库
1. 把本文件夹里的 `本周看板.md`、`本月看板.md`、`总览看板.md`（和本说明）复制进你的 Obsidian 仓库，建议放在一个 `看板/` 文件夹里。
2. Obsidian → 设置 → 第三方插件 → 启用社区插件 → 浏览 → 搜 **Dataview** → 安装并启用。
3. 设置 → Dataview → 打开 **Enable JavaScript Queries**（看板用 DataviewJS，必须开）。
4. 打开任意一个看板即可。

## 看什么
- **本周 / 本月**：顶部统计卡（条数、活跃天、最高产分类）+ 彩色分类分布条 + 按分类分组的条目清单，**点任一条跳回原文**。
- **总览**：近 8 周 / 近 6 月热力表（颜色越深越高产）+ 累计分类排行。

## 调整
- **误收了非分类的日期文件夹**（比如某个也用日期命名的文件夹）：把它名字加进每个看板代码块顶部的 `EXCLUDE` 数组。
- **想换配色**：改 `PALETTE` 数组。
- **加新看板**（如「上周回顾」「某分类专题」）：复制一个看板，改 `collect()` 之后的过滤条件即可。

## 前提
看板依赖条目是 `## HH:MM · 标题` 这种二级标题格式（journal-organizer 写入的就是这个格式）。
